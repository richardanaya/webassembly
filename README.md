# webassembly

A rust library for working with web assembly.

- [x] `#![no_std]` and `alloc`
- [x] includes all opcodes
- [x] primitive type -> wasm type representation helper
- [x] wasm type representation -> primitive type helper

```toml
[dependencies]
webassembly = "0.8"
```
